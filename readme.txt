Designed and built a computer game using Python that emulates the popular Game Pigeon game "Filler"
Implemented the Minimax algorithm with alpha-beta pruning to create a computer opponent that can think four steps ahead, providing a challenging and engaging experience for players
Used the Tkinter library to create an intuitive graphical user interface that includes login/signup functionality and a leaderboard, enhancing the user experience and encouraging competition
